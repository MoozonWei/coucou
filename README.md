# Lin UI

[快速上手]: https://doc.mini.talelin.com/start/

安装模块

```shell
npm install
```

点击**工具**选项，选择**构建npm**，生成miniprogram_npm文件夹

> 配置中引入
>
> ```json
> {
>   "usingComponents":{
>     "l-button":"/miniprogram_npm/lin-ui/button/index"
>   }
> }
> ```
>
> 页面中使用
>
> ```html
> <l-button type="default">默认按钮</l-button>
> ```

项目配置勾选 **增强编译**
